# 🤖 Flux Telegram Bot

Telegram-бот для SaaS **Flux** (защищённое соединение и приватный доступ в интернет). Работает поверх REST API основного сайта (Laravel + Sanctum) и позволяет пользователям управлять аккаунтом прямо в Telegram.

Сделан на **Node.js 20+** с библиотекой [grammY](https://grammy.dev).

## Возможности

- 🔐 **Вход** по email/паролю от аккаунта Flux (бот сразу удаляет сообщение с паролем; токен — read-only).
- 👤 **Аккаунт** — тариф, статус, дата окончания, дни, баланс.
- 🔌 **Подключения** — список профилей и данные для подключения (с тап-копированием).
- 📱 **Устройства** — привязанные устройства.
- 💳 **Тарифы** — актуальные планы с сайта (публичный эндпоинт).
- 🎁 **Рефералы** — реферальная ссылка, код, баланс, число приглашённых.
- 📚 **Документация** — кнопки на пошаговые инструкции по 8 платформам.
- 🆘 **Поддержка** — контакты из настроек проекта.
- 🌐 **RU / EN** — переключение языка на лету.
- 💾 Сессии хранятся в файле (`data/sessions.json`).

## Требования

- Node.js **≥ 20**
- Запущенный сайт Flux с доступным API (`/api/v1`)
- Токен бота от [@BotFather](https://t.me/BotFather)

## Установка

```bash
cd bot
npm install
cp .env.example .env      # на Windows: copy .env.example .env
# впишите BOT_TOKEN и при необходимости API_BASE_URL / SITE_URL
npm start                 # или npm run dev (auto-reload)
```

Проверка синтаксиса без запуска: `npm run check`.

## Переменные окружения

| Переменная | Описание | По умолчанию |
|------------|----------|--------------|
| `BOT_TOKEN` | Токен от @BotFather | — (обязательно) |
| `API_BASE_URL` | База API Flux | `http://127.0.0.1:8123/api/v1` |
| `SITE_URL` | База сайта (для веб-ссылок) | `http://127.0.0.1:8123` |
| `DEFAULT_LANG` | Язык по умолчанию (`ru`/`en`) | `ru` |
| `SESSION_FILE` | Файл сессий | `./data/sessions.json` |

## Используемые эндпоинты API Flux

| Метод | Маршрут | Назначение |
|-------|---------|-----------|
| POST | `/api/v1/auth/login` | Вход → read-only токен |
| POST | `/api/v1/auth/logout` | Отзыв текущего токена |
| GET | `/api/v1/me` | Профиль, подписка, рефералы |
| GET | `/api/v1/connections` · `/connections/{uuid}` | Подключения |
| GET | `/api/v1/devices` | Устройства |
| GET | `/api/v1/plans` | Тарифы (публично) |
| GET | `/api/v1/meta` | Контакты/брендинг (публично) |

## Команды бота

`/start` · `/menu` · `/login` · `/logout` · `/help`

## Безопасность

- Токены, выдаваемые боту, имеют права **только на чтение** (`read`).
- Сообщение с паролем удаляется сразу после обработки.
- Вход защищён rate-limit на стороне API (`throttle:6,1`).
- Выход (`/logout`) отзывает персональный токен на сервере.
